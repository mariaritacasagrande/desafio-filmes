# Desafio Frontend Cubos Filmes


### Pré-requisitos

* ter o git instalado ;
* ter o node instalado;
* ter yarn instalado.

### Execução

1. Abra o Prompt de comando (CMD);

2. Clone este repositório do github:

```
$ git clone 
```

3. Vá para a pasta desafio-filmes:

```
$ cd desafio-filmes
```

4. Instale as dependêcias:

```
$ yarn
```

5. Execute o aplicativo:

```
$ yarn start
```
